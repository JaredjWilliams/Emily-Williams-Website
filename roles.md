# Roles

## CSS
- master
  - Create responsive, maintainable stylesheets using modern CSS features (Grid, Flexbox, Custom Properties)
  - Optimize CSS performance through efficient selectors, minimal specificity, and strategic use of preprocessors
  - Design scalable CSS architectures (BEM, OOCSS, or utility-first approaches) that promote reusability

## HTML
- master
  - Write semantic, accessible HTML5 markup that enhances SEO and screen reader compatibility
  - Structure content hierarchically with proper document outline and ARIA attributes when needed
  - Ensure cross-browser compatibility and validate markup against W3C standards

## TS
- master
  - Design robust type systems using advanced TypeScript features (generics, conditional types, mapped types)
  - Refactor legacy JavaScript codebases to TypeScript while maintaining backward compatibility
  - Create maintainable code with proper interfaces, type guards, and strict type checking configurations

## Coding
- master
  - Write clean, self-documenting code following SOLID principles and design patterns
  - Debug complex issues using systematic approaches and advanced debugging tools
  - Review and refactor code to improve performance, maintainability, and reduce technical debt

## Teaching
- master
  - Break down complex technical concepts into digestible, practical examples
  - Adapt teaching methods to different learning styles and skill levels
  - Create comprehensive documentation, tutorials, and code examples that enable others to learn independently

## Application Architecture
- master
  - Design scalable, maintainable system architectures that balance performance, security, and developer experience
  - Evaluate and select appropriate technologies, frameworks, and patterns for specific project requirements
  - Plan and implement microservices, monoliths, or hybrid architectures based on business needs and constraints

## UI
- master
  - Create pixel-perfect, responsive interfaces that work seamlessly across devices and browsers
  - Implement interactive components with smooth animations, transitions, and micro-interactions
  - Build reusable UI component libraries with consistent design systems and style guides

## Design
- master
  - Develop cohesive visual identities and design systems that communicate brand values effectively
  - Create wireframes, mockups, and prototypes that balance aesthetics with functionality
  - Apply principles of typography, color theory, spacing, and visual hierarchy to create compelling designs

## User Experience
- master
  - Conduct user research, create personas, and map user journeys to inform design decisions
  - Design intuitive information architectures and navigation patterns that reduce cognitive load
  - Test and iterate on designs based on usability testing, analytics, and user feedback to optimize conversion and satisfaction
